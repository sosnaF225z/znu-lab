# 🏗️ Help Guide: Task Management API

Welcome to the Task Management System integration guide. This document provides step-by-step instructions on setting up the SDK and authenticating your requests.

---

## 💾 Installation

To use the Task Management SDK, you must first install the package and its dependencies. Depending on your environment, follow the instructions below.

### Prerequisites
* **Python:** 3.8 or higher
* **Pip:** Package installer for Python

### Steps
1.  **Clone or Download the SDK**
    If you generated the SDK locally, navigate to the generated directory:
    ```bash
    cd ./sdk/python
    ```

2.  **Install via Pip**
    You can install the SDK directly from the local folder:
    ```bash
    pip install .
    ```

3.  **Verify Installation**
    Check if the package is available in your environment:
    ```bash
    pip show task_management_api
    ```
